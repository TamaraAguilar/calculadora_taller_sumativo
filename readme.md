# Calculadora - Taller Sumativo S4

Programa simple en Python que realiza operaciones matemáticas básicas con dos números
(suma, resta, multiplicación, división, módulo y potencia).

## Estructura del Proyecto

```
calculadora/
│
├── calculadora.py
└── operaciones/
    ├── suma.py
    ├── resta.py
    ├── multiplicacion.py
    ├── division.py
    ├── modulo.py
    └── exponente.py
```

## Instalación

1. Clona este repositorio:

```bash
git clone <url-del-repositorio>
```

2. Navega al directorio del proyecto:

```bash
cd calculadora
```

## Uso

Ejecuta el programa principal:

```bash
python calculadora.py
```

El programa te pedirá dos números y mostrará el resultado de todas las operaciones de la siguiente forma:

```
Ingresa el primer número: 10
Ingresa el segundo número: 3
########################################################
Suma: 13.0
Resta: 7.0
Multiplicación: 30.0
División: 3.333333333333333
Módulo: 1.0
Exponente: 1000.0
########################################################
```
